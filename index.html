2026-05-19
Crypto Easy

You get a weird encoded string that starts with an underscore. The only hint is a Java class name. This is NOT normal Base64 — it's Java AltBase64, used inside the JDK to safely store folder names on case-insensitive filesystems.

  • Notice the string only has lowercase letters and weird symbols like ! and quotes

CTF

Caesar Cipher — MinneHack 2026

2026-05-19
Crypto Easy

A Caesar cipher is one of the oldest tricks in the book — just shift every letter by the same amount. Since there are only 25 possible shifts (A→B, A→C, … A→Z), you can literally try them all and read off the one that makes sense. No advanced math needed!

  • You get an encrypted string: something like 'NggnpuRapragPurpx...' — looks like gibberish.
  • You know it's a Caesar cipher, so every letter has been shifted by the same number of positions in the alphabet.
  • There are only 25 possible shifts. Write a quick script to try each one, or use an online decoder.
  • The correct shift reveals readable English text — that's your flag!
  • Flag: flag{ConfuseOrangeObligationParadeAbsentSurprise}
