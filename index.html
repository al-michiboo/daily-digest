BKISC CTF 2026 — Cryptografie

2026-05-19
Crypto Easy

You get a weird encoded string that starts with an underscore. The only hint is a Java class name. This is NOT normal Base64 — it's Java AltBase64, used inside the JDK to safely store folder names on case-insensitive filesystems.

  • Notice the string only has lowercase letters and weird symbols like ! and quotes
  • This is NOT normal Base64 — it's Java AltBase64 from FileSystemPreferences
  • Read the OpenJDK source for that class — it shows the exact alphabet and how it works
